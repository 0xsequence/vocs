# Get Access Keys

If you want to access the RPC endpoint of a mainnet rollup deployed with Conduit, you’ll need to obtain an access key which has, currently, unlimited requests.

## Steps

1. Sign up or log in to [Conduit](https://app.conduit.xyz/)
2. Head to the [Access Keys](https://app.conduit.xyz/access-keys) page
3. Select the mainnet rollup you wish to access
4. Click “Get Access Key”
5. Copy the RPC URL and your access key string
6. Start sending requests to the Conduit rollup RPC endpoint
